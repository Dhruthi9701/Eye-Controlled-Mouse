## üí° Tips for Better Recognition

1. **Speak Clearly**: Wait for the beep before speaking
2. **Use Exact Phrases**: Use the commands listed above for best results
3. **Natural Language**: Many commands accept variations (e.g., "Open" vs "Launch")
4. **Pause Between Commands**: Wait for confirmation before next command
5. **Good Microphone**: Ensure microphone is working and not muted

## üìù Notes

- Commands are **case-insensitive** (you can say them any way)
- Many commands have **multiple variations** (e.g., "Open" = "Launch" = "Start")
- **Browser commands** work in Chrome/Edge
- **System commands** work on Windows
- **TTS feedback** confirms actions (can be disabled)
